The above header is styled using a custom made CSS parser with benefits:

  • Create CSS file ones and deploy them in multiple projects by just adding dynamic fields to them.
  • Maybe you want to use one CSS file to handle multiple theme outputs and color schemes.
  • This way you can keep CSS files on one place without copypasting them and make them more maintenance friendly.
  • Every CSS file is cached to reduce CPU usage.
  • More info and download at: https://github.com/webdesign-benny/phpCSSparser

Next in upcomming version:

  • Create your own dynamic type fields. At the moment keywords "$insert" and "$attribute" are standard.
  • Error and general logfile to inspect the result after parsing.
  • Compress and decrompress functions.
  • Usage of this script true a REST webservice without to install anything.
